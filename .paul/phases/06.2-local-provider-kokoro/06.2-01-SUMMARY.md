## Install Note for End User

To actually run Kokoro synthesis (not just have the GUI show the dropdown):

```
conda activate text2audiobook
pip install kokoro soundfile huggingface_hub
# Windows: install espeak-ng .msi from https://github.com/espeak-ng/espeak-ng/releases
# Linux:   apt install espeak-ng
# macOS:   brew install espeak-ng
```

First Kokoro Start will download the ~500 MB pinned revision to `~/.cache/huggingface` (or `$HF_HOME`).
